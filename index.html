Projects

Enhancing IoT Security Through Anomaly Detection and Intrusion Prevention in Cyber-Physical Systems

Developed an advanced intrusion detection system to enhance IoT security in cyber-physical environments using supervised deep learning models including CNN, LSTM, and DNN architectures. The system was evaluated on benchmark datasets such as KDDCup99, NSL-KDD, and UNSW-NB15, achieving accuracies of 100%, 95.1%, and 87.4% respectively. A Flask-based web application was deployed to demonstrate real-time anomaly detection with standardized data preprocessing. This research was published in the Proceedings of ICCIET 2024 by Atlantis Press.

Keylogger – Capturing Keystrokes

Developed a proof-of-concept keylogger application capable of capturing keyboard input events through a graphical user interface. This project was created for cybersecurity awareness and educational demonstrations, highlighting how keystroke monitoring works and emphasizing the importance of endpoint protection and ethical security practices.

Student Database Management System

Developed a web-based Student Database Management System (SDMS) using PHP, HTML, CSS, and MySQL to manage student records, faculty information, attendance tracking, exam data, and registration details. Implemented full CRUD operations across relational tables and incorporated views, stored procedures, functions, and triggers to maintain data consistency and integrity. The system supports automated registration, attendance validation, and reliable ACID-compliant database transactions.
